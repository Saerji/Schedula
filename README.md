# 🔎 System Overview

Schedula is a system that aims to assist personalities that struggles to access, edit, and update their academic schedules. This system aims to assist its users to organize thair schedules all in one go. Made in compliance to the author's final project in the course CC 102 - Advance Computer programming.

# ✨ Key Features

Schedula has several functions user can utilize:

➣ **Adding** schedule information such as course code, course title, day, time, and room designations.  
➣ **Viewing** all schedule informations in a tabular format.  
➣ **Update** existing schedule informations one by one.
➣ **Delete** existing schedules that may no longer be needed.

# 🛠️ Technologies Used

Schedula utilizes the Python (version 3.14.4) programming language specifically:  
➣ **CustomTkinter** - a modern Python UI library on top of Tkinter  
➣ **Pillow** - an open-source library in Python used for image processing  
➣ **Datetime** - a built-in module in Python used for real time monitoring  
Moreover, the system uses several python programming concepts such as control flow statements, file handling, modular programming, etc.
